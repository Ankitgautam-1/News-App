# News App

A new Flutter project.

## Getting Started

This is simple news app, created to show how i use clean architecture in my flutter code.

I'm using news_api[https://newsapi.org/] to fetch news

### API Key (secure setup)

This project does **not** hardcode the NewsAPI key. Provide it at runtime via `--dart-define`.

Run:

```bash
flutter run --dart-define=NEWS_API_KEY=your_key_here
```
